## Software requirements:

Install the requirements via:
```bash
pip install -r requirements.txt
```

#### Major
- tensorflow 2.4.0
- matplotlib
- numpy
- gdown

#### Minor
- imageio
- pydot

In addition, for visualization purposes install graphviz (https://graphviz.org/)
